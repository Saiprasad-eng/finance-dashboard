# Finance Dashboard UI

## Overview
A clean and interactive finance dashboard built using React and Tailwind CSS. 
It allows users to view financial summaries, transactions, and insights.

## Features
- Dashboard summary (Balance, Income, Expenses)
- Spending breakdown chart
- Trend line chart
- Transactions table
- Role-based UI (Viewer/Admin)
- Insights section
- Responsive design

## Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts

## Setup Instructions
1. Clone the repository
2. Install dependencies
npm install

3. Run development server
npm run dev

4. Open browser
http://localhost:5173
